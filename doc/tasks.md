Danh sách các task cần hoàn thành:

1. Thiết kế và hoàn thiện Game Loop (xử lý sự kiện, hàng đợi, va chạm, cập nhật bản đồ)
2. Bổ sung các API cho bot client (lấy trạng thái, gửi lệnh, đặt bẫy, nghỉ ngơi, thu thập kho báu)
3. Mở rộng event WebSocket/Socket.IO (cập nhật trạng thái, thông báo, xác thực, phân quyền)
4. Hoàn thiện mô hình dữ liệu (bổ sung trường cho Map, Game, Player, trạng thái trận đấu)
5. Xây dựng dashboard trực quan (hiển thị bản đồ, vị trí, kho báu, bẫy, bảng xếp hạng)
6. Bổ sung logic xử lý hành động tàu (di chuyển, đặt bẫy, nghỉ ngơi, thu thập kho báu, va chạm)
7. Cơ chế ghi điểm và xếp hạng (tính điểm, cập nhật bảng xếp hạng, kết quả)
8. Viết test case cho các tình huống (kiểm thử logic, kiểm soát lỗi)
9. Hoàn thiện log, kiểm soát lỗi, bảo trì (bổ sung log, kiểm soát lỗi, dễ mở rộng)
